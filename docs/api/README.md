# API Documentation

## Standards

- All endpoints must be documented with OpenAPI/Swagger.
- Include example requests and responses.
- Document all possible error codes.

## Endpoints

### SCM Engine

- `POST /api/scm/calculate`: Calculate SCM for a given input.
- `GET /api/scm/status`: Get current system status.

### User Management

- `POST /api/auth/login`
- `POST /api/auth/register`
